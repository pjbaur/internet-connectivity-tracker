# Project Context for AI Assistant

## What this project does
The Internet Connectivity Tracker is a Java 21 + Spring Boot application that continuously measures internet connectivity and latency across multiple targets. It performs round-robin TCP probes every second, stores results in Elasticsearch, and provides REST APIs for status, history, and metrics. The stack is fully containerized with Docker and includes optional Kibana dashboards for visualization.
